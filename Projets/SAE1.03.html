Installation d'un poste de développement

Mission :

Installation d'un poste de développement.

But :

Installation de Linux sur une machine virtuelle afin d'y installer un environnement de développement. Création d'une carte mentale pour résumer toute l'installation.

Nombre de participants :

1

Travail que j'ai effectué :

J'ai installé et configuré un système Linux sur une machine virtuelle. Une fois l'installation du système finalisée, j'ai configuré les permissions de super utilisateur afin de pouvoir installer des programmes. J'ai alors installé par packets debian de quoi installer d'autres programmes : snap, flatpak. Il m'a ensuite fallu installer un environnement de développement (Idea Intellij) de plusieurs façons : via flatpak, snap ou encore par archive. J'ai ensuite créé une carte mentale qui résume plus précisément le processus complet des installations. Vous pouvez retrouver cette carte mentale dans les ressources de ce projet afin de la consulter. Le projet comprenait aussi une évaluation séparée qui consistait à traduire des instructions assembleur en anglais.
